Comprehensive Health Tracking

Our system provides real-time monitoring of vital health parameters including heart rate (BPM) and ambient temperature. The data is collected through IoT sensors, transmitted wirelessly, and visualized on a responsive web dashboard.

Heart Rate Monitoring

Accurate BPM measurement using pulse sensor technology with medical-grade precision.

Temperature Sensing

Precise ambient temperature readings with LM35 sensor (±0.5°C accuracy).

Wireless Data Transmission

Real-time data transfer via WiFi using ESP8266 module with secure protocols.
